# book_of_changes
製作易經數字卦占卜程式

## 數字卦占卜方式：

輸入三個三位數數字

將前兩個數字各除以 8，最後一個除以 6

如果整除，則餘數即為除數 8 或 6

各數字算完的餘數照八卦的順序選取卦

八卦順序為乾一、兌二、離三、震四、巽五、坎六、艮七、坤八

乾為天、兌為澤、離為火、震為雷、巽為風、坎為水、艮為山、坤為地
